# Установка зависимостей

Для исправления ошибок TypeScript необходимо установить зависимости:

```bash
cd frontend
npm install
```

После установки все ошибки TypeScript должны исчезнуть, так как будут установлены:
- `@types/react` - типы для React
- `@types/react-dom` - типы для React DOM
- `typescript` - компилятор TypeScript

Ошибки, которые вы видите сейчас, связаны с отсутствием установленных типов React. После выполнения `npm install` они исчезнут.
